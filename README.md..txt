# Rifa iPhone 14

Página simples para venda de números de rifa.

## Funcionalidades
- Seleção de números
- Redirecionamento para WhatsApp
- Pagamento via PIX

## Tecnologias
- HTML
- CSS
- JavaScript

## Como usar
1. Abra o arquivo index.html
2. Escolha um número
3. Finalize pelo WhatsApp
